CSE480/591 Optimization with Metaheuristics

Author: Buğrahan Aydın
Student ID: 20210702018

This project implements a Tabu Search metaheuristic for solving the
Frequency Assignment Problem (FAP).

How to run:
1. Open the project in Eclipse (Java 21).
2. Make sure the 'data' folder is in the project root.
3. Run Runner.java.
4. Results will be printed to the console and/or CSV files.

The algorithm was tested on CELAR and SURPRISE benchmark instances
obtained from the ZIB FAP Repository.
